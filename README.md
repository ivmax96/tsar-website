# tsar-website

Dockerfile и ресурсы для создания недостающего образа. Этот образ упоминается в домашней работе на этом [курсе](https://stepik.org/course/253174/promo), но авторы курса его не добавили, что вводит обучающихся (и меня в том числе) в ступор. Поэтому я решил исправить этот недостаток, так как курс в целом хороший.

Я не только создал этот репозиторий, но и выложил готовый образ на Docker Hub. Чтобы скачать его к себе на компьютер воспользуйтесь командой:

```bash
docker pull ivmax96/tsar-website
```

А затем можно упростить ему имя:

```bash
docker tag ivmax96/tsar-website tsar-website
```

После выполнения этих команд вы получите недостающий образ `tsar-website` на вашей машине. Теперь выполнение ДЗ станет более интересным.

Вы также можете склонировать этот репозиторий и в его корневой директории выполнить команду:

```bash
docker build -t tsar-website .
```
